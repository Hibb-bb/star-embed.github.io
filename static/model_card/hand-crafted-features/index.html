🏗️ Model Architecture

Hand-crafted features represent traditional feature engineering approaches for time series analysis. These features are manually designed based on domain knowledge and statistical properties of astronomical time series data:

  • Approach: Manual feature engineering
  • Features: Statistical, frequency, and temporal features
  • Domain Knowledge: Astronomy-specific characteristics
  • Input: Raw time series data
  • Output: Engineered feature vectors
  • Features Include: Mean, std, skewness, kurtosis, periodicity, etc.
